Welcome to ACHP's documentation!
================================

.. htmlonly::

    ACHP is an open source program written to model direct expansion and secondary loops for residential-scale systems.
    
    It was written by `Ian Bell <mailto:ian.h.bell@gmail.com>`_, during his time at the `Herrick Labs <https://engineering.purdue.edu/Herrick/index.html>`_ of Purdue University.  He can be contacted with any questions or bugs.
    
    **Quickstart**:
    
    * To download the executable for ACHP, head to `ACHP downloads <https://sourceforge.net/projects/achp/files/>`_.
    
    * To download a PDF version of the documentation you see here, `click here <_static/ACHP.pdf>`_.
